# Islands Claude Icon Design

## 目标
- 为扩展提供缺失的 `icon.png`，用于 VS Code 扩展列表显示。
- 风格贴近 Claude：极简、柔和、低对比。

## 设计选择
- 图形方向：抽象图形
- 底形状：圆角方形
- 内部图形：双弧互抱（对话/拥抱意象）

## 规格
- 画布：`512x512`
- 输出：`icon.png`（主文件，建议 256+）
- 底色：`#F2EFEA`
- 主色：`#D97757`
- 圆角：`rx=96`
- 线条：`strokewidth=44`，端点圆角

## 结构说明
- 使用圆角方形底，留足外边距（约 32px）。
- 两条弧线左右对称，朝向中心，形成“互抱”视觉。
- 弧线不贴边，保持呼吸感与留白。
